Description: 
------------
This dataset provides products derived from The Lunar GNSS Receiver Experiment (LuGRE).
It is a public dataset containing raw GNSS observables and signal sample batches acquired by the LuGRE payload throughout Firefly's Blue Ghost Mission 1 (BGM1).

The dataset contains:
- Raw observable data for GPS and Galileo L1/E1 and L5/E5 bandwidth.
- Signal samples for L1/E1 and L5/E5 bandwidth
